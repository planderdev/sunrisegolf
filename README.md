# 썬라이즈 골프클럽 — 리뉴얼 (Warm Hotel Luxury)

부천 도심 인도어 골프(원본: https://sunrisegolf-one.vercel.app/) 랜딩페이지.
**crownparkhotel.co.kr** 스타일(컬러·폰트·섹션 구성)을 반영한 밝은 호텔 럭셔리
정적 사이트. 인터랙션은 cruziana 계열(Lenis·핀 갤러리)을 유지.

## 디자인 방향 (crownparkhotel 반영)
- **컬러**: 화이트 + 웜 크림 섹션(`#F6F3EC`), 니어블랙 텍스트(`#16120B`),
  포인트 **브론즈/토프 `#806344`**(+딥 브론즈 `#A0682B`) — 밝고 따뜻한 호텔 톤
- **폰트**: `Cormorant Garamond`(고대비 세리프 디스플레이) + `Alexandria`(라틴
  라벨/내비) + `Pretendard`(한글 본문) — crownpark의 Alexandria+세리프+Pretendard 구성 반영
- **섹션 구성**: 풀블리드 히어로 → 소개 → **시설(Around Spots형 필터 캐러셀)**
  → 멤버십(오퍼) → 갤러리 → 등록 → FAQ → "CALL NOW" CTA → 다크 푸터
- 시그니처: 세리프 숫자 멤버십 요금표 + 우측/하단 퀵 액션 레일

### 클럽 소개 — crownpark "sec01" GSAP 핀 리빌
중앙 아치형 이미지(share3)가 스크롤에 따라 **풀스크린으로 확장**되고, 양옆 조각
이미지가 슬라이드되며 제목이 **흰색으로 반전**, 하단 콘텐츠 바가 페이드인.
crownpark의 `.pin` + `share1~5` + scrub 타임라인을 그대로 반영. 데스크톱(≥992px)
핀 / 모바일·reduced-motion은 정적 폴백(`.about.is-static`).

### 헤더 · 플로팅 · 아이콘 · 로더
- **헤더 Reservation 버튼**: 전화번호 버튼 → `Reservation`(화살표) 버튼.
  현재 `#membership` 임시 연결, **차후 네이버예약 URL로 교체 예정**(HTML 주석 표기).
- **플로팅 메뉴**: crownpark `#floating` 스타일 — 토글(+) + 더보기 링크(전화/네이버
  예약/네이버지도/카카오지도) + Top 버튼(스크롤 600px↑ 노출, Lenis 스크롤).
- **Remix Icon**: 플로팅·예약 버튼·시설 컨트롤(이전/다음/재생·정지)·링크 화살표 등.
- **Rose Curve 로더**: math-curve-loaders의 `r = a·cos(5θ)`(5장 꽃잎) SVG 트레일
  애니메이션을 프리로더에 적용(회전·호흡·트레일).

### Facilities — crownpark "Around Spots" 패턴
카테고리 필터 탭(ALL/PRACTICE/SCREEN/RELAX) + Swiper 카드 캐러셀 + 하단 컨트롤
(자동재생 진행바 · 이전/다음 · 재생·일시정지). crownpark의 `slideTabWrap` +
`sec03Swiper` + `pagiWrap(autoProgress/prev/play/next)` 구조를 그대로 반영.
- 데스크톱: 가로 핀 갤러리 활성 / 모바일: `gsap.matchMedia`로 핀 해제(스와이프)

## 인터랙션 / 기술 (cruziana 스크롤 인터랙션 반영)
cruziana 스택(Lenis · GSAP ScrollTrigger · fullPage.js · Swiper)을 참고해 반영:
- **Lenis 부드러운 관성 스크롤** — cruziana 특유의 '버터 스크롤' 감각. GSAP
  ScrollTrigger와 동기화하고, 앵커 이동도 Lenis로 부드럽게 처리.
- **핀 고정 가로 스크롤 갤러리** — fullPage.js의 가로 전환 느낌. 데스크톱에서
  세로 스크롤이 갤러리를 가로로 이동(01→06). 모바일은 네이티브 스와이프로 폴백.
- **상단 골드 스크롤 진행바** · 히어로/시설 이미지 **패럴랙스**
- **Pixabay 히어로 비디오**(골프 스윙, 폴백 소스 + 실패 시 그라데이션)
- **AOS** 진입 리빌 · 멤버십 탭 전환 · 카운트업 · 단일 오픈 FAQ ·
  모바일 풀스크린 메뉴 · 프리로더
- 접근성: 키보드 포커스, reduced-motion(스무스/핀/패럴랙스 자동 비활성), 모바일 하단 액션 바
